# minui-miyoo355-dufs-pak

This is a simple [MinUI PAK](https://github.com/shauninman/MinUI/blob/main/PAKS.md) that brings the [dufs file server](https://github.com/sigoden/dufs) to your Miyoo Flip. This tool allows you to transfer files wirelessly and access your device's storage from any web browser on your local network.

## Usage
1. Download [File Server.pak.zip](https://github.com/DendyLusus/minui-miyoo355-dufs-pak/raw/refs/heads/main/File%20Server.pak.zip) and extract it.
2. Copy the File Server.pak folder to your SD card at `/Tools/my355/`.
3. Insert the SD card back into the device and boot MinUI.
4. Make sure the device is [connected to Wi-Fi](https://github.com/DendyLusus/minui-miyoo355-wifi-pak).
5. Toggle File Server through the Tools menu.
6. Open a web browser and type in `[MIYOO-IP-GOES-HERE]:65080` to access the File Server.
